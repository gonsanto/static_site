# **Static site**

## Le **Static Site** est un site web statique qui aide a créer des marques/brands avec le caffé. Il y a différentes sections qui montre l'équipe, les anciens projets, un newsletter et un footer avec toutes les adresses et liens légaux.


# **L'objectif** 
## L'objectif va être de ré-créer au **pixel près**, tout en respectant la résolution de 1920x1080, le site web statique en utilisant la maquette créer par l'équipe design sur figma en **HTML CSS**.
## Le code doit être clair et concis en utilisant html et css modernes (Flexbox et Grid. Éviter les répétitions (notamment en utilisant des classes et des sélécteurs CSS génériques).
